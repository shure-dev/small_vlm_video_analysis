# 手順書（グラウンドトゥルース）— コンロ点検

> ⚠️ **この手順書は評価用の正解であり、モデル（VLM）には一切与えない。**
> モデルは動画のみを入力とし、「いつ・何をしたか」を自力で出力する。
> この手順書は、モデル出力の時系列・内容を人間が採点するためだけに使う。

- **対象**: Takara standard 2口ガスコンロ（Si センサー / 型番 T21EG）の始業前点検デモ
- **動画**: 元動画はGit管理外。公開回帰用には1fps・16枚のモザイク済み抽出フレームだけを同梱し、アプリのpreviewは `data/` に生成する
- **撮影**: 一人称、手元を上から。指差し呼称（安全確認）を含む作業手順のデモ。

## ステップ定義

| # | ステップ | 内容 | 動画で見える手がかり（検出フェーズ用） |
|---|---------|------|------------------------------------|
| 1 | 点火確認 | コンロのつまみを回して**点火し、消火して戻す** | 手が前面のつまみを操作 / 五徳中央に炎 |
| 2 | 指差し確認① | 点火・消火の結果を**指差し呼称** | 人差し指を立てた手（他指は畳む）がコンロを指す |
| 3 | グリル開閉 | **魚焼きグリルを開けて、閉じる** | 前面下部のグリル扉/トレイが引き出される→戻る |
| 4 | 指差し確認② | グリルの状態を**指差し呼称** | 人差し指を立てた手がグリル/コンロを指す |
| 5 | 電池確認 | **点火用電池が2本入っていること**を確認 | 最下部の電池ケースを引き出し、電池が見える |

## 期待される時系列（順序）

```
点火(1) → 指差し(2) → グリル開閉(3) → 指差し(4) → 電池確認(5)
```

代表フレーム（peek, cv2抽出）での裏付け:
- t≈1.5s: つまみに手（ステップ1）
- t≈7.6s: グリルを引き出し中（ステップ3）
- t≈13.8s: 電池ケースを引き出し（ステップ5）

## 採点の観点

1. **カバレッジ**: 5ステップのうち何個を検出できたか。
2. **順序**: 出力された順序が上記と一致するか。
3. **時刻精度**: 各ステップの発生時刻（秒）がどれだけ合っているか（±1〜2秒許容の目安）。
4. **誤検出/幻覚**: 存在しない行動を出していないか。

## 備考・曖昧点

- ステップ5「電池が2本」はユーザー口頭仕様。動画では電池ケースを引き出す動作までが明瞭で、「2本」の本数までVLMが読めるかは未知（要検証）。
- 最終の指差し確認（グリル/電池後）がもう一度ある可能性があるが、口頭仕様には未記載のため本手順書では5ステップとする。実映像に応じて追記する。
