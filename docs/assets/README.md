# Demo asset attribution

`factory_ego_temporal_grounding.gif` is a downsampled screen recording of the local annotation workspace. It plays ten clips back to back at increased speed, derived from [`builddotai/Egocentric-10K`](https://huggingface.co/datasets/builddotai/Egocentric-10K).

- Upstream dataset: Egocentric-10K
- Upstream publisher: builddotai
- Upstream license: Apache-2.0
- Access: gated; contact information is required by the upstream host
- Local changes: clip selection, temporal sampling, downscaling, application UI, and human/model timeline overlays

The repository's MIT license applies to the application code, not to upstream dataset media. Use and redistribution of this GIF and the source media remain subject to the upstream license, access terms, and privacy requirements.
