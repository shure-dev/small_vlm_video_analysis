# 人手イベント定義ガイド

イベントは、動画内で観察できる一つの動作です。動画を見ずにイベント候補を作らないでください。

## 粒度

- 一つの動詞を中心に書く
- 道具や対象物は、境界判断に必要な範囲で具体化する
- 長い工程は複数の観察可能な動作へ分ける
- 瞬間的すぎて境界を再現できない概念は避ける
- 状態を記録する場合は、開始と終了を目視で合意できるものに限る

良い例:

- 作業者が金属部品を箱に入れる
- 作業者が操作盤の緑色のボタンを押す
- 作業者が布の端を両手で揃える

避ける例:

- 作業をしている（曖昧）
- 次工程の準備をしている（意図の推測）
- 品質が良い（観察可能な動作ではない）

同じ動作が繰り返されたら、同じevent IDへ複数区間を付けます。日本語文の英訳はannotation完了後の派生工程で行い、event IDを変えません。
